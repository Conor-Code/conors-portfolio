I thought a newsletter would be a cool idea,  I’m aiming for one per week. 
I know I’ll probably be talking into the void for a while, but I think it will be a good way to document my projects. 


## This week

Anyway, I’m stacking a lot of TODOs on top of each other this week. 
So I’ve decided on 2 things I need to focus on. This portfolio and Program-Pit. Once those are shipped, I can focus on the real goal. 

## My main goal

Building in public means shipping before it's perfect. So I’m going to lean into that.
I needed a base, so I built this portfolio page, now I need an app.
 
I came up with the idea for The Daily Ship out of my own frustrations. I’m a dev that’s in the early days of his grind trying to make a living online. And I know without a social media presence, that grind is 10x harder. That said this X grind is not for the weak, you can’t miss a day. So how can I solve both of these problems at the same time?

## The Daily Ship

The goal of my content is to provide value.
So what if I built a tool that takes inspiration from what I’m building, allows me to personalise ideas how I please, and pulls in the latest tech trends to stay relevant.


Then it combines all of that information into daily post ideas and threads.
I could edit and format the post how I’d like. Then simply add the post straight into X. 

It would be set up to give hourly/daily notifications to keep me motivated to post without the hassle of having to create content from scratch everyday, on top of developing.

In a nutshell. This is The Daily Ship. This is what I’m building towards.
Now I’m no Saas expert, literally, this is my first SaaS. But my thinking is if it really fixes a problem for me, then I expect it to do the same for others.

Hopefully by the next letter I have some real progress. 
Lots more of this to come. Build logs, stuff I’ve learned, I just needed to give some context about where I’m at right now to begin.
